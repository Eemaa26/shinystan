Installation instructions for the latest version of shinyStan

Latest version: 1.0.0 (25 Feb 2015)

Step 1. Open R (or RStudio) or restart if already open.

If you don't have the devtools package install it by running 

install.packages('devtools')

Step 2. Load the devtools package and install shinyStan by running 

    library(devtools)
    source_url("https://github.com/stan-dev/shinystan/raw/develop/install_shinystan.R")
    install_shinystan()

If you prefer, you can download (or copy) the install_shinystan function here and run install_shinystan().

Step 3. After installation finishes restart R.

You can then load the shinyStan package with library(shinyStan).
